Select

Directive for creating a material designed select input. There are three variants that can be used:

  • The full fledged control, based on mdcMenu.
  • A control based on the native select html element. This control may provide better usability on mobile devices.
  • A control based on the native select multiple html element.

As for all Blox Material directives: the controls can be used with or without the angular forms package.

Accessibility

This directive will automatically apply proper accessibility attributes to the control, the contained menu and its items
